# Promo Console Design

This is a local operator console, not a marketing page.

- Keep the first screen task-focused: account state, generation, publishing, and logs.
- Use compact panels with clear labels and stable controls.
- Prefer restrained contrast, readable form fields, and dense but not crowded spacing.
- Avoid decorative hero sections, illustrations, gradients, or nested cards.
- Treat logs as a first-class surface. Command output should be readable and easy to copy.
